# Lógica JS

Repositório com exercícios de lógica de programação utilizando JavaScript.

O objetivo deste projeto é manter a prática com JavaScript puro e exercitar raciocínio lógico, especialmente em momentos em que o foco principal está em frameworks como React.

Os exercícios são implementados em JavaScript e executados diretamente no navegador.

## Tecnologias Utilizadas

- HTML5
- JavaScript (ES6+)

## Objetivo

- Praticar lógica de programação
- Manter fluência com JavaScript puro
- Exercitar resolução de problemas
- Evitar dependência excessiva de frameworks

## Como Executar

1. Clone o repositório:

```
git clone https://github.com/mateushahndev/logica-js.git
```

2. Abra o exercício desejado no navegador.

3. Os resultados podem ser visualizados no **console do navegador**.

## Observações

Este repositório é destinado apenas para prática e estudo pessoal.  
Novos exercícios podem ser adicionados conforme a evolução dos estudos.
